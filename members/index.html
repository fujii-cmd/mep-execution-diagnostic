1. 全体整理

この業務は、診断単体で終わらず、改修戦略、実施設計、発注支援まで伸びる構造を持っています。

Layer 1：診断

現況診断、ΔT診断、実COP診断、搬送動力診断、制御成立診断。

Layer 2：改修戦略

改修方針、システム転換案、概算比較、更新優先順位。

Layer 3：更新設計

機械、電気、制御を一体で扱う実施設計と発注支援。

一般には分断されやすい3層を一体化できる点が、この業務の競争優位です。

2. 空調アルゴリズム（ΔT診断を核に）

入口は「ΔTが小さい」「冷房が効かない」。そこから測定信頼性、流量、熱交換、末端制御、混流、熱源、系統不均衡へ掘り下げます。

入口：ΔTが小さい（例：5℃未満）または冷房が効かない ↓ STEP1：測定値の信頼性確認 STEP2：流量の把握 STEP3：熱交換成立確認（コイル） STEP4：末端制御（VAV/バルブ） STEP5：混流の有無 STEP6：熱源側 STEP7：系統全体の不均衡 → 原因確定
分類主な原因構造改修方針期待効果
① ΔTシックネス型過流量 / 三方弁 / バイパス / ポンプ過大変流量化、二方弁化、差圧制御ΔT改善 / ポンプ電力削減
② VAV未成立型風量絞れない / 再熱多発 / センサー位置不良制御復活 + 風量適正化快適性向上 / ファン電力削減
③ 搬送過大型ポンプ・ファン過大搬送最適化 / INV導入電力削減 / 機器寿命延長
④ 熱源側問題能力不足 / 制御不良熱源更新 or 構成転換COP向上 / 安定性向上

3. 給排水衛生 実効診断

温度・滞留・水理・衛生リスクを同時に扱い、給湯・給水・排水まで改修計画に接続する統合モジュールです。

主診断指標

  • 貯湯温度 / 往き温度 / 返り温度
  • 給湯循環温度マップ
  • 水齢リスク評価 / 末端到達時間
  • レジオネラリスク指数（LRI）
  • 給湯配管熱損失量

見抜きたい問題構造

  • 循環バランス不良
  • デッドレグ・長水齢
  • 保温不足・断熱欠損
  • 過大ポンプ・絞り運転
  • 排水不良・通気不良

4. 電気設備の診断アルゴリズム

受変電・配電・動力・INV負荷を含む電気設備全体を、容量・品質・バランスの観点で診断します。

① 高調波リスク型

THD過大、発熱、設備寿命低下。

② 相不平衡型

相電流偏り、中性線過負荷。

③ 容量不適型

トランス・幹線・盤余裕不足または過大。

④ INV集中型

負荷集中、品質悪化、拡張制約。

5. MEP横断の改修優先順位

停止リスク、安全リスク、省エネ効果、保全性、更新難易度、将来拡張性を共通軸として並べます。

6. 3案提案テンプレート

案①現状踏襲、案②性能改善、案③システム転換の3案比較を標準フォーマットにします。

案① 現状踏襲更新

既存システム維持、老朽機器更新。工事影響は小さいが根本改善は限定的。

案② 性能改善型

既存活用 + 制御改善。効果と工事性のバランスが良い推奨軸。

案③ システム転換型

DOAS + FCU / 放射 / 水系VRFなど抜本見直し。高効率だが初期費・工事影響大。
